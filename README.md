RecipeSwap 🍳  
  
RecipeSwap არის თანამედროვე Android აპლიკაცია სტუდენტებისთვის, რომელიც საშუალებას გაძლევთ იპოვოთ, შეინახოთ და მართოთ სხვადასხვა რეცეპტი. შექმნილია Jetpack Compose-ითა და MVVM არქიტექტურით.  
  
ძირითადი ფუნქციები

* მომხმარებლის ავტორიზაცია  
* პროდუქტების რეცეპტების დათვალიერება (TheMealDB API)  
* რეცეპტების ძებნა სახელით  
* რეცეპტების გალერეა  
* პორციების ავტომატური გადათვლა  
* ფავორიტების შენახვა  
* თანამედროვე ინტერფეისი
